Spam Detector For SMS Messages

A Machine Learning App That Explains It All

Please write a nice message... or not:


Prediction times vary.

For best accuracy, try SMS (Short Message Service) text messages between 2 and 400 characters in length.
The model was trained on an SMS Spam Collection which can be downloaded at the UCI Machine Learning Repository.
This labelled data is comprised of 747 spam and 4825 ham (legitimate) text messages.
